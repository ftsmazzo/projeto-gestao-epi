# Indice da Base do Projeto

## Como usar

Para iniciar o projeto no Cursor:

1. Crie ou abra o repositorio local.
2. Copie esta estrutura documental para o repositorio.
3. Leia `README.md` e `docs/CURSOR-CONTEXTO.md`.
4. Execute primeiro o `Prompt 00` se o repositorio ainda nao tiver documentacao.
5. Execute o `Prompt 01` somente quando for hora de criar a fundacao tecnica.
