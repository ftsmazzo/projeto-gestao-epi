# Prompt 00 - Criar Repositorio Local com Documentacao

Use este prompt se o repositorio ainda nao existe no computador do usuario.

```markdown
Crie apenas a estrutura documental inicial do projeto de Gestao Digital de Entrega de EPI.

Objetivo:
Preparar um repositorio local para receber a documentacao de arquitetura, Blueprints e prompts antes de iniciar codigo.

Crie a seguinte estrutura:

- README.md
- docs/CURSOR-CONTEXTO.md
- docs/blueprints/
- docs/prompts/
- docs/deploy/
- docs/decisions/

Inclua os Blueprints e documentos fornecidos pelo usuario nesta estrutura, sem alterar o conteudo conceitual.

Regras:
- Nao implementar codigo ainda.
- Nao instalar dependencias.
- Nao escolher stack definitiva se ela ainda nao estiver aprovada.
- Nao criar app, API, banco, Docker ou migrations nesta etapa.
- Fazer commit e push apenas se o usuario ja tiver conectado o repositorio remoto.

Ao final, responda exatamente:
## Resultado
## Arquivos alterados
## Como testar
## Testes executados
## Pendencias
## Commit
```
